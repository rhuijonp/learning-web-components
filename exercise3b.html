Different HTML elements allow for giving structure to a web document, this is better appreciated on web pages that have defined areas for a kind of content.
An example of such a page is a news website, where there is area for advertisment, the main content, or maybe the top stories of the day lined up on one side of the page.
HTML probably has an element for any kind of need a developer may have as to how some content should be displayed, some times a list is required, and you have ordered or unordered list elements, or you want to show some content on a grid,like sports statistics, then you may go with a table/row/col elements,etc.
Maybe you need to put a menu on a website and link the options to local or remote files. The options are unlimited, and without a doubt, there is an HTML element that facilitates getting your job done.
